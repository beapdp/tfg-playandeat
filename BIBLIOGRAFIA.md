# 📚 Bibliografía y Recursos de Consulta: Play & Eat

Este documento recopila todas las fuentes de información, librerías y herramientas consultadas para el desarrollo técnico de este Trabajo de Fin de Grado.

---

## 💻 Documentación Oficial (Tecnologías Core)

1.  **Next.js (Framework Principal)**
    *   *Fuente:* [https://nextjs.org/docs](https://nextjs.org/docs)
    *   *Uso:* Consulta de App Router, API Routes y Server Components.

2.  **Supabase (Base de Datos y Auth)**
    *   *Fuente:* [https://supabase.com/docs](https://supabase.com/docs)
    *   *Uso:* Configuración de tablas, políticas RLS y gestión de autenticación JWT.

3.  **Tailwind CSS (Diseño y Estilos)**
    *   *Fuente:* [https://tailwindcss.com/docs](https://tailwindcss.com/docs)
    *   *Uso:* Guía de clases de utilidad para el diseño responsivo y sistema de colores.

4.  **TypeScript (Tipado Estático)**
    *   *Fuente:* [https://www.typescriptlang.org/docs/](https://www.typescriptlang.org/docs/)
    *   *Uso:* Definición de interfaces y tipos para garantizar la seguridad del código.

5.  **Leaflet (Motor de Mapas)**
    *   *Fuente:* [https://leafletjs.com/](https://leafletjs.com/)
    *   *Uso:* API de mapas interactivos de código abierto.

6.  **React-Leaflet (Puente React)**
    *   *Fuente:* [https://react-leaflet.js.org/](https://react-leaflet.js.org/)
    *   *Uso:* Implementación del mapa como componentes de React.

7.  **OpenStreetMap (Cartografía)**
    *   *Fuente:* [https://www.openstreetmap.org/](https://www.openstreetmap.org/)
    *   *Uso:* Proveedor de las imágenes (tiles) y datos geográficos del mapa.
